css o hojas de estilo en cascada

sirven para formatear el diseño de una o varias paginas a la vez

con ellas controlas el color, la fuente, el tamaño del texto, el espacio entre los elementos

como están dispuestos, que imagenes o o colores de fondo son usados

diferentes despliegues para diferentes tamaños de pantallas, dispositivos


en cascada significa que si un estilo es aplicado a un elemento padre

tambien es aplicado a los elementos hijos derivados del padre

quiere decir si se ajusta el color del body a blue, todos los h, p y demas seran azules

a menos que se especifique lo contrario

css puede implementarse de 3 formas

en linea

usando el atributo style dentro del elemento html

interna

usando la etiqueta style dentro del elemento html head

externa

usando la etiqueta link dentro del elemento html head

La forma comun de utilizar css es de forma externa

para ello está el archivo styles.css dentro de este repositorio (archivo de extension .css)

con una hoja externa, puedes cambiar la apariencia de todo un sitio web editando solo un archivo
